Welcome to the Primary Care Workforce portal. We have set this up as a place where you can submit applications and returns in an efficient way.  

At the moment you can use the portal for applications and submissions for the New to Partnership Payment Scheme (N2PP) and the Additional Roles Reimbursement Scheme (ARRS) but we may incorporate wider schemes in the future.

We aim to make the use of the portal as straightforward as possible, if you have any comments please let us know by emailing england.newtopartnershipenquiries@nhs.net
